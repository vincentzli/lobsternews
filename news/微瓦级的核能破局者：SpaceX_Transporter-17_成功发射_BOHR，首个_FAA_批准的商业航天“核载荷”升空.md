# 微瓦级的核能破局者：SpaceX Transporter-17 成功发射 BOHR，首个 FAA 批准的商业航天“核载荷”升空

2026年7月7日，SpaceX的猎鹰9号火箭从范登堡太空军基地腾空而起，执行Transporter-17拼车发射任务。在众多被送入轨道的载荷中，一颗名为BOHR（Betavoltaic Orbital High-Reliability）的1U立方星（CubeSat）显得平平无奇。这颗由佛罗里达州City Labs公司开发的卫星，外表与高校或科研机构的普通轨道器并无二致，但它的发射却在人类航天史上写下了里程碑式的一笔：这是全球首个进入太空的商业核动力卫星载荷。

过去六十余年里，空间核动力技术一直是国家级航天机构的专属特权。诸如旅行者号（Voyager）、新视野号（New Horizons）以及好奇号火星车（Curiosity）等深空探测器，无一例外依赖着以钚-238为燃料的放射性同位素温差发电机（RTG）。这些系统虽然能输出数百瓦的热功率（再通过温差电偶转化为电能），但其自重大、政治敏感度极高，且造价动辄数千万美元。而City Labs正在展示一种颠覆性的替代方案：固态纳米氚（NanoTritium™）辐伏电池。该技术直接利用氚的β衰变产生连续的电能。

需要指出的是，BOHR卫星并非整星完全依赖核能驱动；它依旧使用标准的太阳能电池板来维持基本系统运转，而纳米氚辐伏电池则作为独立的实验载荷运行，以验证这种微型核电源在轨道极端环境下的实际工作性能。

```
       [ 氚衰变 ]
          │
          ▼ (β粒子：平均 5.7 keV)
   ┌────────────────────────────────┐
   │ InGaP 宽禁带半导体结            │ <── 无晶格位移损伤
   └────────────────────────────────┘
          │
          ▼ (直接转换)
      [ 持续电流 ] ──> [ 备用 / 常开传感器 ]
```

#### 直接β转换的物理学法则
与通过塞贝克效应（Seebeck effect）将衰变热转化为电能的RTG不同，辐伏电池（Betavoltaics）是非热转换器件。City Labs的纳米氚电池在物理机制上更接近光伏发电（太阳能电池）。其基本原理是，氚（$^3\text{H}$）衰变为氦-3，同时释放出一个反中微子和一个β粒子（即高能电子）：

$$^3\text{H} \to ^3\text{He} + e^- + \bar{\nu}_e$$

氚的半衰期为12.3年，这决定了其具备一条极易预测的、缓慢衰减的输出功率曲线。其释放的β粒子平均动能为5.7 keV（峰值可达18.6 keV）。在半导体材料中，晶格位移损伤的能量阈值通常在100 keV以上。这意味着这些低能电子在穿过半导体晶格时，根本不会产生令高能辐射源头疼的晶格原子位移缺陷（即无晶格损伤）。

传统的硅半导体在低载流子产生率下会出现严重的暗电流（漏电）问题。为此，City Labs采用了宽禁带（WBG）III-V族化合物半导体，具体为磷化铟镓（InGaP）。其约1.9 eV的宽禁带极大降低了逆向饱和电流，从而在微瓦级功率尺度下，依然能将开路电压（$V_{oc}$）和转换效率最大化。此外，氚燃料本身被安全地锁定在固态金属氚化物晶格中，彻底规避了高压气体泄漏的风险。

#### 跨越监管的“行政迷宫”
事实上，BOHR任务最艰巨的突破可能不在于技术，而在于行政审批。在航天历史上，发射任何核材料都需要国家元首级或多部门联合的高规格审批。而BOHR成为了首个成功跑通美国联邦航空管理局（FAA）商业核载荷发射审批流程的项目，该流程依据《第20号国家安全总统备忘录》（NSPM-20）设立。

City Labs为此编制了一份详尽的发射安全分析报告。该报告通过了美国能源部国家核安全局（NNSA）直属的桑迪亚国家实验室（Sandia National Laboratories）的独立审查与验证。桑迪亚实验室的背书为FAA最终发放发射许可证扫清了障碍，也为未来的商业航天核能任务树立了标准化的监管范本。

#### 微瓦vs兆瓦：规模化落地之争
这颗微瓦级“核心脏”的升空，也在航天科技界引发了一场关于辐伏电池商业前景的激烈辩论。支持者看重其长达20余年的超长寿命以及极端的温度耐受性（可在-248°C至150°C的温区正常工作），这使它成为熬过寒冷漫长的月夜或探测永久阴影坑的终极电源解决方案。

City Labs首席执行官彼得·卡鲍伊（Peter Cabauy）强调：“这次的创新不仅在技术层面，更在监管层面。空间核能应用已经存在了几十年……但要真正走向下一步并实现规模化，它必须走向商业化。”

然而，硬币的另一面是技术的规模化瓶颈。Founders Fund合伙人、Varda Space Industries联合创始人德里安·阿斯帕鲁霍夫（Delian Asparouhov）等风险投资人指出，由高算力太空站和轨道太空工厂驱动的商业近地轨道经济，动辄需要千瓦级（kW）甚至兆瓦级（MW）的电力支撑。微瓦到毫瓦级的辐伏电池在这些“吞电兽”面前杯水车薪。

```
功率尺度：
┌────────────────────────────────────────────────────────┐
│ 辐伏电池 (微瓦级 - 毫瓦级)                              │ <-- BOHR 载荷
├────────────────────────────────────────────────────────┤
│ RTG同位素温差发电机 (瓦级 - 百瓦级)                    │ <-- 火星车
├────────────────────────────────────────────────────────┤
│ 太阳能电池阵列 / 裂变反应堆 (千瓦级 - 兆瓦级)            │ <-- 国际空间站
└────────────────────────────────────────────────────────┘
```

此外，埃隆·马斯克（Elon Musk）此前也多次对“核-电”直接转换概念嗤之以鼻，坚定地捍卫“太阳能+储能电池”的架构，并将太阳比作宇宙中“免费的核聚变反应堆”。

尽管受限于微瓦级输出，当前的辐伏电池只能退居幕后，充当备份系统、实时时钟、加密密钥存储以及低功耗遥测传感器的安全网。但在深空探索或长达14天的严酷月夜里，当太阳能电池板完全失效时，这种辐伏电池能以涓流充电的方式为电容器蓄能，或为维持卫星生存的加热器供电，从而避免航天器冻结损坏。BOHR迈出的第一步，正在向行业证明：商业航天电源的未来，完全可以跨越太阳的边界。
