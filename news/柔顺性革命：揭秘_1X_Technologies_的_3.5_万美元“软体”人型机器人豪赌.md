# **柔顺性革命：揭秘 1X Technologies 的 3.5 万美元“软体”人型机器人豪赌**

**人型机器人终于走出了工厂实验室，但其形态与我们预想中的“钢铁之躯”大相径庭。当特斯拉的 Optimus 和 Figure 还在死磕金属骨架与“刚性优先”的工业视觉时，1X Technologies 已经带着 NEO 杀入战场。这台套着“针织衫”外壳、奉行“触觉安全”至上哲学的机器人，正试图重新定义机器人学的底层架构。**

**机械结构的范式转移：腱驱动对决齿轮箱**
当前机器人行业正处于一场关于“齿轮”还是“绳索”的技术路线斗争。传统人型机器人高度依赖刚性谐波减速器。这种结构虽然精密，但具有极高的“反向驱动惯量”——这意味着一旦与人发生碰撞，它不会弯曲，只会折断。1X 首席执行官 Bernt Børnich 对这种“古典主义”路线的评价极具挑衅性：在日常生活中使用 100:1 减速比的机器人，无异于在你的关节上绑着哑铃。

NEO 选择了基于生物学的“腱驱动系统”（Tendon-driven system）。通过将 Revo1 电机置于躯干，利用柔性聚合物腱绳驱动四肢，1X 实现了高达 95% 的反向驱动性。这种“柔顺优先”的物理架构让 NEO 能够表现出如同肌肉般的弹性。这也是为什么 NEO 可以穿着柔软的软衬衫而非生硬的机甲，它是一台旨在让人拥抱，而非让人避让的机器。

**HomeOS 与“幻影”经济学**
NEO 的大脑建立在一个“远程示教转自主学习”的闭环之上。在 HomeOS 架构下，1X 采用了一种名为“Ghosting”的方法：人类操作员远程驱动机器人执行精细任务，如拿取玻璃杯或折叠丝绸。这不仅是远程办公的变体，更是一个庞大的“数据工厂”。每一次远程操作都在训练底层的“大行为模型”（LBM），推动 NEO 向家务劳动的“零样本”自主化迈进。

然而，NEO 为视觉导航配置的双 8.85 MP 摄像头所带来的“全时监控”属性，正引发激烈的隐私争论。尽管 Sam Altman 与 OpenAI 将 1X 视为通用人工智能（AGI）在物理世界的最后一块拼图，但社交平台上的质疑声从未停歇：一个在私人住宅中“看”到一切的机器人如何保证安全？1X 的方案是本地处理和物理状态指示灯——当有人在背后“Ghosting”时，机器人的耳环会亮起。

**社会经济冲击：3.5 万美元的“时间机器”**
以 3.5 万美元为目标定价，NEO 被定位为一种对“家庭时间”的资本投资。如果一台机器人能接管 80% 的家务劳动，家庭劳动力的估值将一夜之间发生重构。我们正在从“电器时代”迈入“具身劳动”时代。

但技术层面的长效性仍是悬在头顶的达摩克利斯之剑。相比 Figure 或波士顿动力那些额定工作数千小时的工业级执行器，人造腱绳在高强度家庭环境下的疲劳寿命仍是未知数。NEO 会成为人型机器人的“iPhone 时刻”，还是每隔 500 小时就需要进厂“换弦”的精密乐器？随着量产开启，硅谷精英与中产阶层都将见证这场豪赌的结局。
