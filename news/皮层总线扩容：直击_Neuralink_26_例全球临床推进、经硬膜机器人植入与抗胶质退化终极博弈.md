# **皮层总线扩容：直击 Neuralink 26 例全球临床推进、经硬膜机器人植入与抗胶质退化终极博弈**

将侵入式脑机接口（BCI）从定制化的实验室学术样品，推进为标准化、规模化的医疗级通用硬件，这一历史性临界点在 2026 年年中正式到来。Neuralink 的全球临床试验版图已扩展至三大不同监管体系下的 26 名人类患者：从作为主阵地的美国 **PRIME** 研究，到加拿大 **CAN-PRIME**（以多伦多西方医院为核心），再到英国 **GB-PRIME**（覆盖伦敦大学学院医院与纽卡斯尔），以及即将在阿布扎比克利夫兰诊所启动的 **UAE-PRIME** 临床试验。

曾经，这是一项充满工匠色彩的手术：神经外科医生需要经历数小时的开颅剥离硬脑膜、手动作图绘制血管布局，并使用定制微型操纵器逐根植入。而如今，它正在被重塑为一种自动化、极具高可重复性的机器人标准作业。然而，随着 Neuralink 的 N1 植入物从单人英雄主义迈向多中心临床试验，工程团队的核心挑战也正在发生深刻转移——从单纯的手术执行力，转向跨越数年的神经生理稳定性、信号解码安全性，以及针对严重运动功能障碍与肌萎缩侧索硬化症（ALS）康复的监管规模化落地。

### 手术范式革命：经硬膜机器人植入

高通道数神经接口长久以来的最大瓶颈，始终是手术层面的物理摩擦。人类大脑外层的保护性屏障——硬脑膜（Dura mater），是一种坚韧的纤维膜，其厚度大约是 Neuralink 柔性聚酰亚胺电极丝（宽度约 5–8 微米）的 10 至 20 倍。在传统神经外科手术中，放置微电极通常需要进行手动的**硬脑膜切除术（Durectomy）**或**硬脑膜切开术（Durotomy）**——即切开并剥离硬脑膜，从而将运动皮层的软脑膜表面完全裸露出来。

然而，硬脑膜切除术会带来巨大的临床风险：脑脊液（CSF）渗漏、直接皮层损伤、感染概率暴增，以及长期的硬膜下组织粘连。2026 年 5 月，在多伦多西方医院进行的一台 CAN-PRIME 手术中，Neuralink 首次公开部署了其定制 **R1 手术机器人** 的重磅升级功能：**自动化经硬膜植入（Automated Transdural Insertion）**。

```
[ 传统硬脑膜切除术 ]
颅骨切开 -> 手工剥离硬脑膜 -> 裸露皮层表面 -> 电极丝植入 -> 脑脊液渗漏与高创伤风险

[ R1 经硬膜植入 ]
颅骨切开 -> 保持硬脑膜完整 -> 膜下 OCT/光学传感 -> 微针高弹高速穿刺 -> 低创伤电极丝精准定位
```

为了在完全不切开硬脑膜的前提下，将 64 根柔性电极丝上的 1,024 个电极直接“穿刺”跨越硬膜且不撕裂膜下血管，Neuralink 对 R1 机器人的植入头与计算机视觉算法构架进行了彻底重构：
1. **膜下血管 3D 映射**：R1 集成了光学相干断层扫描（OCT）与多波长光学传感器，能够在半透明至不透明的硬脑膜下方，对皮层表面的微血管进行实时成像，实时计算出分辨率达 10 微米级的软脑膜毛细血管 3D 深度图。
2. **高动力学植入针机制**：R1 的植入针由直径小于 30 微米的钨铼合金细丝制成，它能够以极高的瞬时加速度穿透硬脑膜，将超柔性聚酰亚胺电极丝精确推入目标皮层深度（M1 区第 4/5 层，深度约 1.5–2.0 毫米），并在毫秒级时间内迅速退针，使柔性电极丝平稳锚定而不发生机械弯折。

正如 Neuralink 首席神经外科医生 Matthew MacDougall 博士在技术简报中所述：*“消除了硬脑膜切除步骤后，开放性神经外科手术被简化为一种微米级的精准微创针刺。我们将手术时长压缩到了 45 分钟以内，同时大幅降低了脑脊液污染与感染风险。”*

### 通道物理学、BPS 指标与“退丝危机”的工程终局

脑机接口的硬通货永远是信息吞吐量，在行业标准的目标获取网格测试（Grid Tasks）中，这一指标被量化为**每秒比特数（Bits Per Second, BPS）**。更高的 BPS 意味着更流畅的多轴光标控制、极速数字打字体验，以及低延迟的机械肢体操控。

```
+------------------------+-----------------------------------------------------------+
| 指标 / 特性            | Neuralink N1 规格与性能数据                               |
+------------------------+-----------------------------------------------------------+
| 总通道数               | 1,024 个电极，分布于 64 根柔性电极丝上                   |
| 电极丝几何结构         | 厚度 5–8 µm，定制聚酰亚胺材质，配有铂/铱触点              |
| 外壳与无线遥测         | 气密封装钛合金圆盘（23mm x 8mm），定制 2.4GHz / BLE        |
| 峰值目标获取速率       | ~8.0 至 10.5 BPS（算法优化后）                             |
| 采样频率               | 单通道原始数据 ~20 kHz；板载尖峰分选 / 频域分箱           |
+------------------------+-----------------------------------------------------------+
```

2024 年初，当首位患者（Noland Arbaugh）接受 N1 植入时，设备初期展现出了极高的 BPS 速率，但随后遭遇了严重的性能衰减：超过 85% 的电极丝从脑组织中发生了“脱出”（Retraction）。这一问题的根源在于脑部的微观机械运动——在正常的血管搏动和头部旋转过程中，大脑在颅骨内部存在 1 至 2 毫米的动态偏移。由于 N1 的钛合金外壳是嵌入并固定在颅骨上的，而早期电极丝仅植入皮层 3 至 5 毫米且缺乏足够的应力缓冲缓冲带，脑组织的位移直接将电极丝拉出了皮层通道。

面对这一危机，Neuralink 的硬件与软件工程团队在第二位患者（Alex）以及后续 CAN-PRIME 和 GB-PRIME 临床试验中，部署了软硬协同的“双重对策”：
- **深层皮层锚定与应力缓冲**：电极丝的植入深度从原来的 ~3–5 毫米大幅增加至最深 8 毫米，使记录点深入皮层更深层次的同时，形成了一条更长的机械“系绳”（Tether），从而有效吸收大脑的微观位移。
- **动态尖峰分选（Spike-Sorting）自适应算法**：针对部分发生微小位移的通道，Neuralink 升级了板载与主机端的信号解码管线，解码策略从单单元尖峰分选（Single-unit spike sorting）全面转向多单元群集率编码（Multi-unit population rate coding）。

通过重新调校解码器对低幅局部场电位（LFP）和背景神经元群集活动的敏感度，Neuralink 成功将 Noland 的 BPS 性能恢复至 10.0 以上，甚至超越了他脱丝前的基线水平。而在患者 2（Alex）及后续所有参与者中，电极丝脱出现象已被基本遏制。

### 长期神经生理学：无法绕过的“胶质瘢痕墙”

尽管机械层面的脱出危机已在手术与工程端被暂时遏制，但 Neuralink 乃至整个神经科技领域，依然面临着一个尚未攻克的生物学终极障碍：**慢性异物炎症反应与反应性胶质增生（Glial Scarring）**。

当人工合成材料被长期植入神经组织时，静息状态的星形胶质细胞和小胶质细胞会被激活并发生反应性胶质增生。在 6 到 24 个月的时间跨度内，这些细胞会在聚酰亚胺电极丝周围形成一层致密且绝缘的胶质瘢痕包囊。

```
[ 第 0 个月：新鲜植入 ]
微电极触点 <---> 神经元膜直接接触（高信号幅值 ~150µV，低阻抗）

[ 第 12-24 个月：反应性胶质增生 ]
微电极触点 <--- [ 胶质包囊 / 反应性星形胶质细胞 ] ---> 远端神经元（信号大幅衰减 <20µV，高尖峰阻抗）
```

1. **阻抗暴涨**：随着胶质包囊不断增厚，接口阻抗会从最初的 ~100 kΩ 飙升至数兆欧（MΩ），导致信噪比（SNR）断崖式下跌，许多动作电位被彻底掩埋于系统噪声背景之下。
2. **通道慢性衰退**：在数年的时间尺度上，薄膜聚酰亚胺衬底的生物腐蚀、铂黑电极触点的溶解，加上生物包囊作用，将不可避免地导致通道的累计失效。

加州大学旧金山分校（UCSF）神经外科主任 Edward Chang 博士在社交平台 X 上直言这一材料学硬约束：*“自动化的机器人植入解决了第 0 天的精准度问题，但它无法欺骗基础神经免疫学。脑机接口的长期生命力，完全取决于涂层化学成分以及机械顺应性是否能与大脑皮层极其柔软的组织模量相匹配。”*

为缓解慢性胶质瘢痕，Neuralink 目前正在重点攻关生物活性水凝胶涂层、缓释抗炎药物的电极衬底，以及模量进一步降低、能够完美匹配大脑皮层切变模量（~1 至 3 kPa）的亚微米级超柔性电极丝。

### 数据隐私、神经解码器与监管路线图

随着 Neuralink 的临床步伐横跨加拿大卫生部（Health Canada）、英国 MHRA 以及美国 FDA，监管机构的审查重心正在从急性手术安全性，向长期数据隐私保护与无线遥测安全性转移。

#### 运动意图解码 vs. 被动思想隐私
N1 芯片通过无线方式将数字化后的神经发放率流式传输至体外处理器（平板电脑或电脑）。生物伦理学家与监管机构正在对**神经遥测数据（Neural Telemetry）**的边界展开激烈辩论。尽管当前的解码算法被严格限定在解码“运动意图”（例如通过构想手部运动来驱动光标），但高密度的皮层电极阵列在物理上不可避免地会捕捉到更广泛的群体神经动力学信息，包括认知准备状态、情绪唤起，甚至内部语言矢量。

在 Reddit 的 r/neurotech 板块上，关于原始信号调度的讨论异常火热：*“如果一家商业脑机接口公司把原始的 20kHz 神经数据持续上传云端用于模型训练，他们记录的绝不仅仅是点击事件——他们是在将患者大脑最底层的电生理底座进行永久存档。”*

对此，Neuralink 副总裁兼联合创始人 DJ Seo 在 X 上回应了安全质疑，澄清了系统架构设计：*“N1 圆盘发出的每一个遥测数据包，都采用了硬件级的 AES-256 加密与非对称设备配对。原始神经电信号阵列完全在客户端设备本地进行计算；在对外传输前，解码后的运动意图矢量就已经剥离了所有微伏级的原始元数据。”*

#### 监管里程碑：从 FDA IDE 迈向 PMA
要想将 N1 正式推向严重运动功能障碍与 ALS 康复的商业化市场，Neuralink 必须完成从试验性设备豁免（IDE）可行性研究，向关键性预市场批准（PMA）阶段的跨越：
- **安全性基准**：设备相关的手术部位感染率为零，年均通道失效率低于 2%，并在 5 年以上的周期内证明生物相容性。
- **有效性指标**：维持稳定的目标获取速度（>8 BPS），并为患者带来实质性的独立生活能力提升（如自主交流、电脑操控、电动轮椅驾驶等）。

### 硅谷与神经科技界的技术共识

对于 Neuralink 的推进速度，科技界领袖与学术界呈现出既惊叹于其工程迭代效率、又对长期临床现实保持审慎的微妙态度。

埃隆·马斯克（Elon Musk）在 X 上展望道：*“临床规模从 2 例扩展到跨越 4 个国家的 26 例，证明了 R1 机器人自动化可以将复杂的开脑手术转变为一种门诊手术。高带宽神经接口最终将让‘人机带宽对等’成为现实。”*

相反，前 FDA 神经科技负责人、科罗拉多大学博尔德分校教授 Cristin Welle 博士在近期研讨会上指出：*“Neuralink 的经硬膜微针系统是一项精湛的硬件工程。但 Health Canada、MHRA 和 FDA 等监管机构更关注 3 到 5 年后的信号保留曲线。要想真正推广至数万名患者，必须证明这些通道在人体组织中浸泡三年后，不会坠入一片黑暗的沉默。”*

知名风投家 Marc Andreessen 则给出了更具市场视角的分野观点：*“脑机接口正在经历移动电话在 1990 年代所经历的相同轨迹——从概念炒作走向基础设施建设。谁能率先将手术机器人与软件解码器 API 规范化、标准化，谁就能赢下人类增强（Human Augmentation）这一全新时代的基础平台层。”*
